РАСКРАСКА-ИГРА
==============

Как играть:
  1. Открой файл index.html в любом браузере
     (Chrome, Edge, Firefox, Safari) — просто двойной клик
     по файлу, либо перетащи его в окно браузера.
  2. Сверху выбери картинку: Котик / Ракета / Цветок.
  3. Внизу выбери цвет в палитре.
  4. Нажимай на части картинки — они закрашиваются.
     • Можно закрасить и фон (нажми на пустое место вокруг рисунка).
     • Лепестки, уши, окошко и т.п. красятся по отдельности.
  5. Кружок в "клеточку" в палитре — это ластик (красит в белый).
  6. «Очистить» — сбросить текущую картинку.
  7. «Скачать PNG» — сохранить раскрашенную картинку себе.

Работает офлайн, ничего устанавливать не нужно. Подходит для
телефона, планшета и компьютера.

Папка pechat/ — те же три картинки в SVG (пустые контуры),
если захочется распечатать и раскрасить на бумаге.
